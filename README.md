# 🌱 WasteWise AI - Chennai Waste Segregation Guide

**Your Smart Waste Segregation Assistant for Chennai**

WasteWise AI is an AI-powered web application designed to help Chennai residents properly segregate household waste according to Greater Chennai Corporation guidelines. The platform provides multiple ways to identify the correct waste bin through an intelligent chatbot, searchable waste database, and interactive visual guide.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/saisanthanalakshmi007/wastewise_ai.git
cd wastewise_ai
```

### Run Locally

Using Python:

```bash
python -m http.server 8000
```

Using Node.js:

```bash
npx http-server
```

Open:

```text
http://localhost:8000
```

Or simply open `index.html` in your browser.
