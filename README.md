# 🌱 WasteWise AI - Chennai Waste Segregation Guide

**Your Smart Waste Segregation Assistant for Chennai**

WasteWise AI is a comprehensive web application designed to help Chennai residents properly segregate their household waste according to Greater Chennai Corporation guidelines. The app provides multiple ways to identify which colored bin to use for different types of waste.

## 🎯 Features

### 1. 🔍 Smart Search
- **Real-time search** with autocomplete suggestions
- **Instant results** showing the correct bin with color-coded cards
- **Fuzzy matching** to handle typos and variations
- Database of **200+ common waste items**

### 2. 💬 Interactive Chatbot
- **Conversational AI** that understands natural language
- **Friendly responses** in 2-3 sentences with bin colors
- **Quick action buttons** for common queries
- **Typing indicators** for realistic chat experience
- Handles greetings, questions, and follow-ups

### 3. 📊 Visual Guide
- **Color-coded bin cards** for easy reference
- **Comprehensive lists** of items for each category
- **Visual examples** with emojis for quick identification
- **Hover effects** for interactive experience

### 4. 📚 Educational Content
- **Chennai-specific** waste management information
- **Best practices** for waste segregation
- **Corporation rules** and guidelines
- **Contact information** for help and support
- **Environmental impact** facts

## 🗑️ Bin Categories

### 🟢 Green Bin - Wet/Organic Waste
- Food scraps and leftovers
- Fruit and vegetable peels
- Tea bags and coffee grounds
- Garden waste and leaves
- Egg shells and bones

### 🔵 Blue Bin - Dry/Recyclable Waste
- Paper and cardboard
- Plastic bottles and containers
- Metal cans and tins
- Glass bottles and jars
- Old clothes and textiles
- E-waste (phones, cables)

### 🔴 Red Bin - Hazardous Waste
- Batteries (all types)
- Expired medicines
- Chemicals and pesticides
- Paint and solvents
- CFL bulbs and tubes
- Cosmetics and nail polish

### ⚫ Black Bin - Sanitary/Reject Waste
- Diapers (baby and adult)
- Sanitary napkins and pads
- Soiled tissues and wipes
- Used gloves and masks
- Bandages and cotton
- Personal hygiene items

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or dependencies required!

### Installation

1. **Download or Clone** the repository:
```bash
git clone https://github.com/yourusername/wastewise-ai.git
cd wastewise-ai
```

2. **Open the application**:
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   ```

3. **Access the app**:
   - Open your browser and navigate to `http://localhost:8000`
   - Or directly open the `index.html` file
